# Hello World Plugin

A simple Hello World plugin for WordPress that demonstrates basic plugin development.

## Features

- Displays "Hello, World!" using a shortcode.
- Shows an admin notice in the WordPress dashboard.

## Installation

1. Upload the `hello-world-plugin` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the `[hello_world]` shortcode in your posts or pages.

## Usage

- **Shortcode**: `[hello_world]`
- **Admin Notice**: Automatically displayed in the WordPress admin dashboard upon activation.
